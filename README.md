# IT-312 – Web Applications Engineering  
## Yum Yum Website Project

This project was developed as part of the **IT-312: Web Applications Engineering** course.  
The goal of the project is to practice and apply the basics of **front-end web development** using HTML, CSS, and JavaScript.

---

## Project Description
**Yum Yum** is a simple static website created to demonstrate fundamental web development concepts.

Through this project, we practiced:
- Structuring web pages using HTML
- Styling and layout using CSS
- Adding interactivity using JavaScript

The project focuses on understanding how different web technologies work together to build a functional and user-friendly website.

---

## Technologies Used
- **HTML** – for page structure and content
- **CSS** – for styling and layout
- **JavaScript** – for interactivity and basic logic

---

## Features
- Multiple web pages
- Clean and organized page structure
- Styled layouts using CSS
- Basic interactive elements using JavaScript
- User-friendly design

---

## Learning Outcomes
This project helped in understanding:
- The structure of web applications
- Separation of content (HTML), style (CSS), and behavior (JavaScript)
- Basic front-end development workflow
- Writing clean and readable web code
